우선순위 큐 자료구조의 활용 예시들

  • 다음과 같은 알고리즘에서 활용될 수 있다.
    • 프림 알고리즘
    • 다익스트라 알고리즘 : 도로 교통망 같은 곳에서 나타날 수 있는 그래프에서 꼭짓점 간의 최단 경로를 찾는 알고리즘이다.
    • A* Search algorithm : 주어진 출발 꼭짓점에서부터 목표 꼭짓점까지 가는 최단 경로를 찾아내는 그래프 탐색 알고리즘 중 하나이다.
    • 힙 정렬 알고리즘
  • 데이터 압축을 위한 허프만 코드
  • 운영 체제에서의 로드 밸런싱과 인터럽트 핸들링, 작업 스케줄링

정리사항

  • 힙을 배열로 구현하는 이유 : 배열, 링크드리스트 둘 다 구현할 수 있지만 배열로 구현했을 때 가장 효율적인데, 그 까닭은 새로운 노드를 힙의 마지막 위치에 추가하는 것이 편리하기 때문이다.
  • 힙으로 우선순위 큐를 구현하는 이유 : 배열이나 링크드리스트로 구현하면 최악의 경우, 데이터를 삽입하거나 삭제하는 과정에서 해당 위치를 찾기 위해 저장된 모든 데이터와 우선순위의 비교를 해야 한다.
  • 우선순위 큐와의 차이점 : 힙은 우선순위 큐에 어울리는 완전 이진 트리이며, 힙이 우선순위 큐 자료구조를 구현하는데 활용되는 것 뿐이다.
